Yes. It uses your real Telegram account (not a bot token), so it has access to every channel you're subscribed to — including paid and private ones.

What if my signal format is unusual?

Mistral AI reads signals the way a human would. As long as the message contains a clear buy/sell instruction with price levels, it will parse it correctly — no matter the format.

Is my Telegram account safe?

The bot only reads messages from the channel you configure. It never sends messages, never accesses other chats. Your session file stays on your own computer.

No. Setup takes about 15 minutes. You fill in a simple config file with your credentials and run one command. Full step-by-step guide included.
